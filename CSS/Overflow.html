Overflow Property in CSS

Overflow: visible

This is an example of overflow set to visible. The content that exceeds the boundaries of this box will be fully visible and not clipped.

Overflow: Hidden

This is an example of overflow set to visible. The content that exceeds the boundaries of this box will be fully visible and not clipped.

Overflow: Scroll

This is an example of overflow set to scroll. The content that exceeds the boundaries of this box will be clipped, but scrollbars will be added to allow users to scroll and see the hidden content.

Overflow: Auto

This is an example of overflow set to auto. If the content exceeds the boundaries of this box, scrollbars will be added automatically. If the content fits within the box, no scrollbars will be shown.

overflow-x and overflow-y properties:

Overflow-X: Scroll and Overflow-Y: Hidden

This is an example where horizontal overflow is set to scroll and vertical overflow is set to hidden. The content that exceeds the horizontal boundaries of this box will have scrollbars, while the content that exceeds the vertical boundaries will be clipped and not visible.

Overflow-X: Hidden and Overflow-Y: Scroll

This is an example where horizontal overflow is set to hidden and vertical overflow is set to scroll. The content that exceeds the horizontal boundaries of this box will be clipped and not visible, while the content that exceeds the vertical boundaries will have scrollbars.
